# RenovaGrow - Digital Marketing Agency Web App

Live Site: https://renova-grow.vercel.app

A modern, high-performance agency website built for RenovaGrow,
a digital marketing agency based in Bangladesh. Designed to showcase
services, portfolio case studies, and collect client inquiries.

## Features

- Hero section with animated 3D campaign mockup
- Service case studies with real performance metrics
- Filterable portfolio grid (Videos, Photo Designs, Ad Run, SEO, Web Dev)
- AI-powered contact form with Gemini Blueprint mode
- Dual inbox system: Standard Inbox and Gemini Blueprint
- Contact form stores submissions and redirects to email client
- Fully responsive design

## Services Showcased

- Short-form marketing videos
- Premium photo and ad designs
- Target ad spend campaigns
- Authority SEO
- Custom workflow automations

## Tech Stack

- Vite + TypeScript
- Node.js backend (server.ts)
- Vercel deployment
- Gemini AI integration
